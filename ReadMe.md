# RK3576 YOLOv11检测追踪计数项目

本项目包含基于RK3576开发板的YOLOv11目标检测、追踪与计数完整解决方案，可扩展为RK35XX全系列，推荐RK3576/RK3588。

## 项目结构

### 1. [RK3576AI相机](RK3576AI相机/)
相机硬件相关说明文档，包含硬件配置、接口定义及使用说明，这部分主要是硬件相关，根据自己的硬件配置，**这部分只是针对我的硬件设备进行仓库备份，跟代码无关，其他硬件也可以使用本仓库代码**。

### 2. [yolov11track](yolov11track/)
RK3576开发板板载部署代码，实现目标检测、ByteTrack追踪、计数统计及下位机通讯功能。
- **部署系统**: Debian
- **功能**: 检测、追踪、计数、TCP Modbus通讯

### 3. [yolov11convertrknn](yolov11convertrknn/)
YOLOv11模型格式转换代码，将训练好的模型转换为RK3576可部署的RKNN格式。
- **部署环境**: Ubuntu 20.04
- **功能**: 模型转换（不涉及训练和推理）

### 4. [yolov11train](yolov11train/)
YOLOv11模型训练与推理验证代码。
- **部署环境**: Windows 10 + RTX 3080
- **功能**: 模型训练、推理验证

## 工作流程

1. 在 `yolov11train` 中训练模型
2. 使用 `yolov11convertrknn` 转换模型为RKNN格式
3. 将转换后的模型部署到 `yolov11byteTrack` 在RK3576开发板上运行
4. 参考 `RK3576AI相机` 配置硬件连接
